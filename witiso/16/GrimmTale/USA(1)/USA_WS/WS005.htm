Hi. This is Nancy Beardsley and Ray McDonald. These are the words and phrases from the English USA worksheet for Lesson 5. First, a phrase that we use when we leave（道别用语）:

　

Goodbye.

再见。

　

Next, three nouns used to describe some workers（名词）:

　

attendant

服务员

　

controller

机场管制员

　

assistant communications director

通信部副部长

　

Here is a pronoun used in Lesson 5（本课出现的代词）:

　

both (two persons or things)

两者（两人或物）
　 　 　

Example: Both are reporters.

两人都是记者。

　

Finally, these phrases（短语）:

　

Please write it for me.

请帮我写下它。

　

Please write your name for me.

请把你的名字写给我。
