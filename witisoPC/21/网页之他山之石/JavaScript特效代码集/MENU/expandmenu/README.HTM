下载文件后展开，有一个images目录，里面的图片是所需的所有LOGO，如果你有更好的换掉即可。

（1）index.htm是FRAME帧页面，定义帧页面的布局。没有什么特别的。本例左侧页面left.htm，右侧right.htm，左侧Frame名字“left”，右侧Frame名字“Right”。

（2）menu.js文件不必有任何改动。

（3）改动最大的就是left.htm（左侧帧页面），第一眼你一定会头昏的，其实代码非常有规律，只要改好一层的菜单，其余层的复制/粘贴，再稍微小小改动即可。

下面是一层菜单(父目录级，即最高级)的完全代码(我是从代码中搬下来的，没有任何改动)：

<script language="JavaScript1.2" src="menu.js"></script>

<!--上面引用外部javascript文件，放在BODY标签后面-->

需要改动的地方：黄色的3处是最重要的。第一层文件夹分别用 KB1Parent， KB1， KB1Child。第二层文件夹分别用 KB2Parent， KB2， KB2Child。第几层就改成几，依次类推，复制粘贴后首先改这3处。

其次，绿色的部分，你不必改，只要明白，parent代表父目录，点击它打开下面的子目录。child代表子目录。一个父目录下可以添加很多子目录。

后面的三小段代表3个子目录，只要修改连接URL，状态栏提示信息和页面打开的目标帧即可。target用“_top”在整个页面打开，用 “_blank” 在新的页面打开，用frame的名字“right”在右侧的帧打开。
