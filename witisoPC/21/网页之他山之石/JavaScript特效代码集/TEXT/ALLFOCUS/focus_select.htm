聚焦与文本选择脚本


看看，不仅聚焦在上面的文本框，而且已经将内容选择了。

在<body>区加上如下代码：

<body onLoad="this.document.form1.a.focus(); this.document.form1.a.select();">
<form name=form1>
<input type="text" size="100" name="a" value="哈哈">
<input type="text" size="100" name="b" value="http://www.newhua.com">
</form>

说明：form1是表单的名字，a是上面的文本框的名字，b是下面的文本框的名字。
