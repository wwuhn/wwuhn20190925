自动页面打印脚本

以下代码加入<HEAD>区：

<SCRIPT LANGUAGE="JavaScript">
<!--//
function printPage() {
if (window.print) {
agree = confirm('本页面包含重要信息。\n强烈建议您打印备份！\n\n点确定打印?');
if (agree) window.print();
}
}
//-->
</script>

别忘记：在<BODY>中不要忘记加上一句 <body onload="printPage()"> ，这样才能激活它啊。

用 \n 表示换行。别滥用这个脚本，否则会招人嫌的，如果你的站点的某些页面提供一些重要信息，或者当访问者填写完表单后有一些重要信息，例如用户帐号，密码等数据需要打印，用它最好。
