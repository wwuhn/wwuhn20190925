免登陆脚本-不仅仅适用于网易虚拟社会啊。

以下是网易虚拟社会北京站代码：

<html>
<head>
<title></title>
<meta http-equiv="Content-Type" content="text/html; charset=gb2312">
</head>

<body onLoad="document.nopassword.submit();">

<form name="nopassword" method="post" action="http://bj.netease.com/cgi-bin/login">
<input type="hidden" name="city" value="beijing">
<input type="hidden" name="user" value="你在北京站用户名">
<input type="hidden" name="pass" value="你在北京站的密码">

</form>
</body>
</html>

以下是网易虚拟社会广州站代码：

<html>
<head>
<title></title>
<meta http-equiv="Content-Type" content="text/html; charset=gb2312">
</head>

<body onLoad="document.nopassword.submit();">

<form name="nopassword" method="post" action="http://club.netease.com/cgi-bin/login">
<input type="hidden" name="city" value="guangzhou">
<input type="hidden" name="user" value="你在广州站用户名">
<input type="hidden" name="pass" value="你在广州站的密码">

</form>
</body>
</html>

说明：分别将以上两段代码作为2个文件wy_beijing.htm和wy_guangzhou.htm，运行后直接进入网易虚拟社会北京站和广州站。

原理：本例具有通性，适用于web邮箱（Hotmail），论坛，留言板等。

凡是类似的登陆脚本，都通过用户输入或选择一些项目，然后通过“action”来交给服务器响应处理。你需要做的就是：

（1）首先，将登陆页面保存下来，查看源代码，找到执行的cgi命令，即“action=...”，将它记录下来。

（2）查看源代码看看递交需要哪些项目（例如：用户名ID，密码，E-mail等），将他们的“name”记录下来。

（3）按照上面的格式写代码，填加所有的递交需要的项目，

<input type="hidden" name="name" value="你的注册值"> ，name指“（2）中记录的name，你的注册值指对应的值”。关键是name一定要与登陆页面中需要的一样，不要随便起。

（4）用“（1）中记录的”action 项代替上面代码中的“action=...”即可。

以上的例子是自动运行进入，另一种方法是去掉<body>中的onload，在<form>...</form>中填加一行：

<input type="submit" name="submit" value="递交"> 即可。先填好内容再连线递交。

现在你就可以轻松地做任何事情了。尤其适用于你经常进入的登陆页面。
