自己设置的脚本运行错误信息反馈样式

是否遇见过脚本错误的情况？由于访问者使用的浏览器并非都支持页面里的脚本，所以这种情况经常出现。一般会弹出一个非常难看的脚本运行错误警告窗口，灰蒙蒙的非常难看。

使用这个脚本，你完全可以控制这个窗口的外观样式，甚至里面的说明文字。

将如下脚本加入HTML的<HEAD>区最前面即可，不要在里面加入任何其它的代码。你可以自由改编外观样式，背景色和显示的文字内容。

<script>

function errorbox(){
errorwindow=window.open("","","width=250,height=125")
errorwindow.document.write('<title>脚本错误报告</title><center>该页面发现了脚本运行错误，请通知管理员。<Br><form><input type="button" value="关闭窗口" onClick="window.close()"></form></center>')
errorwindow.document.close()
errorwindow.document.bgColor="#fff7f7"
return true
}
window.onerror=errorbox
</script>

本页面故意使用了一个错误的脚本，代码如下，很明显脚本不是封闭的，少一个“）”

<script>
alert("该脚本错误，没有封闭。"
</script>
