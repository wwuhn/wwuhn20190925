可以使你的网站加入别人的频道和收藏夹
并且作为浏览器默认页的脚本


（将本站加入频道，只适用于IE）

【将本站添加到你的收藏夹】

【下载并运行，让本站成为您的首页】

（1）加入channel的方法：将如下连接指向你的频道文件*.cdf。

<a href="javascript:window.external.addChannel('jschannel.cdf')">

*.cdf文件的内容如下，自己修改以适应自己的站点情况：==============================================================================================
<?XML VERSION="1.0" ENCODING="GB2312"?>

<CHANNEL HREF="http://www.sina.com" SELF="http://www.sina/jschannel.cdf">
<Title>新浪</Title>
<Abstract>最大的新闻网</Abstract>
<LOGO HREF="http://www.sina/images/jschannel.gif" STYLE="IMAGE"/>
</CHANNEL>
（注意：代码中所有地址都为假想，请替换为您所需的地址）

=============================================================================================

（2）加入收藏夹的方法：

<a href="javascript:window.external.AddFavorite('http://www.sina.com/', ' 新浪')">

（3）作为浏览器首页的方法：

制作*.reg文件，下载并运行即可。*.reg文件内容如下：（可用记事本打开编辑）==============================================================================================
REGEDIT4

[HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main]
"Start Page" = "http://www.sina.com"

[HKEY_CURRENT_USER\Software\Netscape\Netscape Navigator\Main]
"Home Page" = "http://www.sina.com"

==============================================================================================
