窗口聚焦控制脚本

测试：打开两个小窗口

 

我们都有这样的经验：打开多个窗口时，同时只有一个窗口是处于“聚焦状态”，或者说是处于“活动状态”，后打开的窗口总是处于聚焦状态。

能否让先打开的窗口或者任意一个比较重要的窗口聚焦呢？这对于你可能是非常重要的，因为该窗口内可能有你最想或最先让访问者看到的内容。那么，这里提供的脚本是非常简单而实用的。

看看我是如何作的：
首先，把下面的脚本加入页面(本页)HTML的<head> 区，控制弹出窗口的文件名和大小。本例先后弹出2个窗口。

<script language="javascript">
function openpopup(){
window.open("focuswin.htm","","width=200,height=200")
window.open("focuswin2.htm","","width=200,height=100")
}
</script>

然后，将下面一行加入<body>区，呼叫窗口。注意：不呼叫上面的代码没有任何动静。这里用了一个小技巧“#”，因为<a href=...>并非指向一个真正的连接。用onclick事件呼叫上面定义的函数。

<a href="#" onclick=openpopup()>测试：打开两个小窗口</a>

下面的脚本是加入到任意想要聚焦的窗口的<head>区的：

<SCRIPT LANGUAGE="JavaScript">

<!-- Begin
var time = 10; // 时间控制（秒），每隔10秒就聚焦。
function remind() {
this.focus();
var real = time * 1000;
setTimeout("remind()",real);
}
remind();
// End -->
</script>
