固定在任意角落的站点LOGO (适合IE4和NS4)

看看左上角的LOGO，位置始终是固定的。

首先下载js脚本文件，将它拷贝到任意目录，然后这样引用外部js脚本(指向js脚本):
<script src="http://yoursite.com/staticlogo3.js"></script>

以上代码加入HTML的<body>区。 不要放在任何封闭的标签内即可。(例如<table>内 )

下载js文脚本：
staticlogo.js (logo在右下角)
staticlogo2.js ( logo在左下角)
staticlogo3.js ( logo在左上角)
staticlogo4.js ( logo在右上角)

说明：用任何文本编辑器打开js文件，修改代码中的图片文件，与你的LOGO一致即可。此 脚本适用于IE和NS浏览器，如果使用IE浏览器还可以设置增强功能，即可以设置LOGO显示的时间， 超过时间就关闭图片显示。
