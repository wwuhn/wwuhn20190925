邮箱地址合法性检测

填一个非法的邮件看看

如下代码加入HTML的<HEAD>区：

<script language="javascript">

/* 检测邮件地址是否为空，是否含有"@"，是否有足够的字符，以及是否有 .net, .gov, .org 或者.com?如果非法提出警告 */
function emailCheck() {
txt=document.myform.email.value;
if (txt.indexOf("@")<3){

alert("邮件地址有问题。请检测是否有'@'");
return false;
}

if ((txt.indexOf(".com")<5)&&(txt.indexOf(".org")<5)
&&(txt.indexOf(".gov")<5)&&(txt.indexOf(".net")<5)
&&(txt.indexOf(".mil")<5)){

alert("邮件地址有问题。"
+"请检查是否含有"
+".com,.net,.org,.gov或.mil");
return false;
}

}
</script>

如下代码加入HTML的<BODY>区，生成表单：

<form name="myform" action="mailto:yourmail@mailserver.com" enctype="text/plain" method="post" onsubmit="return emailCheck()">

<input type="text" name="email" size=40>
<input type="submit" name="submit" value="开始递交">
</form>

注意：表单的名字和文本框的名字要保持前后一样。
