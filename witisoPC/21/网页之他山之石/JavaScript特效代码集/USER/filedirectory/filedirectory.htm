分割显示文件确切路径，明明白白俺的心。
可以访问上级目录了。

将如下代码加入HTML的<body>区即可：

<script language="JavaScript">
<!-- Begin

}
i=s.length-1;
path+="<A HREF=\""+href.substring(0,href.indexOf(s[i])+s[i].length)+"\">"+s[i]+"</A>";

</script>
