显示当前页的路径
当前页:

前言：现在许多朋友都喜欢用“帧”来安排页面的布局。诚然，使用FRAME的确使页面比较易于安排，但是不容忽视的一个问题是：许多初上网的朋友往往直接保存了FRAMESET页面，离线后看时却什么也没有。其实一个含有“帧”的页面，仅仅是一个框架，并非真正的页面，除非你将帧指向的所有页面都保存下来。

对于一个为用户考虑的webmaster来讲，网上的资料就要大家共享，否则你放在网上做什么？(有版权的除外)。所以页面要易于保存才行，人家花了大量的时间在你的站里下载了许多感兴趣的页面，拿回去后确发现是一大堆框架文件，下一次谁还再来呢？所以我很少使用框架来组织自己的网站。

下面的这个脚本就是对那些爱使用frame的朋友提供的，这个脚本能显示当前页面的路径(url)，这样访问者就可以按照这个URL下载它了。

如下代码加入HTML的<HEAD>区：
<script language="JavaScript">
<!-- Begin
var dir = location.href.substring(0,location.href.lastIndexOf('?')+1);
var email = location.href.substring(dir.length,location.href.length+1);
// End -->
</script>

如下代码加入HTML的<HEAD>区：
<script language="JavaScript">
<!-- Begin
document.write(email);
// End -->
</script>
