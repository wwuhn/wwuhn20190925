浏览器检测脚本

检测访问者使用的浏览器是IE还是NS，然后根据检测结果带领他们去相应的页面，可以有效地避免浏览器的不兼容导致的页面浏览异常。

代码非常简单，将如下代码加入HTML的<head>区：

<SCRIPT LANGUAGE="JavaScript">
<!-- Start Code

var ver = navigator.appVersion;

if (ver.indexOf("MSIE") != -1)

{

window.location.href="IE.htm"

}else

window.location.href="NS.htm"

// End Code -->

</SCRIPT>
