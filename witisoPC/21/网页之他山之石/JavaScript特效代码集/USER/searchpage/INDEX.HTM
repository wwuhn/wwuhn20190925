非常酷的JS搜索脚本
－反馈查询结果象专业的CGI程序

测试：检索关键字“java”或“javascript”或“脚本”看看结果的输出。

检索要诀：

1、输入的关键字用空格隔开；
2、选择合适的检索方法(and/or/exact)。

由于IE的一些奇怪的原因，我在调试这个脚本时的确遇到了一点小麻烦，IE不支持离线搜寻(offline search)。因为这个脚本用到了类似下面的一些语句：
search.htm?javascript

search.htm?javascript%20perl&or

search.htm?javascript%20perl&and

search.htm?javascript%20perl&exact

只好又装了NETSCAPE，所以你今后在离线状态下研究这个脚本时，请用Netscape浏览器执行，在线(online)时用IE没有任何问题。
